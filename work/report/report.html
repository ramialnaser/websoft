An overall analysis concludes common improvements such as:

- Keep request count low and transfer sizes small

- Values of id attributes should be unique

- Stop using insecure javascript libraries

- Add a passive flag to every event listener that Lighthouse identified.

- Add a meta description element to the head of each of your pages.

- Add accessible names to links

- Optimizing image sizing

- Ensure text remains visible during webfont load

- Use HTTP2 instead of HTTP1.1

- Remove unused CSS
